### Inventory Management System

### Overview

This repository contains my Inventory Management System assignment developed in Python. The program demonstrates the use of Python programming concepts to manage inventory records efficiently through a command-line interface.

The system allows users to add, view, search, update, and remove product records while also providing inventory statistics and low-stock reports.

### Features

- Add new products to the inventory
- View all inventory records
- Search for a product
- Update product quantities
- Remove products from the inventory
- Display inventory statistics
- Generate low-stock reports
- Menu-driven command-line interface
  
### Technologies Used
- Python

### Concepts Demonstrated
- Function
- Dictionaries
- Conditional statements
- Loops
- Exception handling
- User input validation

### Purpose

This assignment was completed to strengthen my understanding of Python programming and demonstrate the implementation of a simple inventory management system using fundamental programming concepts.
